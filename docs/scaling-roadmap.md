# Scaling Roadmap

## Phase 1

- Single-business demo and staging support.
- Mock adapter and one SalonFlow tenant.
- Server-mediated realtime voice sessions.

## Phase 2

- Multi-business routing.
- Shared cache for stable business context.
- Dedicated observability pipelines.

## Phase 3

- Queue-based tool execution for load spikes.
- Regionalization and tenant isolation.
- Automated regression evaluation in CI.
