# Three-Minute Demo Script

1. Open the receptionist web app and start a voice session.
2. Ask for a service, such as a haircut.
3. Provide a preferred date and time.
4. Choose one of the offered appointment slots.
5. Give name and phone number.
6. Confirm the booking.
7. Show the booking confirmation and tool status trail.

## Talking points

- Low-latency voice experience.
- Real availability lookup.
- No direct database dependency on SalonFlow.
- Mock mode works without production access.
